# DevLinks - Página de Links Pessoais

#Link do portfolio -> https://nebulosaorion.github.io/Portfolio/

Este projeto é uma página de links pessoais, inspirada no estilo do Linktree, que permite centralizar links de redes sociais em um único lugar com um design limpo e responsivo.

## ✨ Demonstração

A página exibe:

- Uma foto de perfil
- Nome do usuário
- Botão para alternar o modo claro/escuro
- Links para redes sociais (Instagram, GitHub, LinkedIn)
- Ícones sociais estilizados

## 📁 Estrutura do Projeto

```
📦 projeto/
├── index.html          # Estrutura principal da página
├── style.css           # Estilo da página
├── script.js           # Lógica para alternar o tema
└── assets/
    └── avatar.png      # Imagem de perfil
```

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- [Ionicons](https://ionic.io/ionicons) para os ícones
- Google Fonts (Fonte Inter)

## 🔧 Funcionalidades

- Alternância entre tema claro e escuro com clique no botão
- Links externos com `target="_blank"` para abrir em nova aba
- Ícones interativos das redes sociais
